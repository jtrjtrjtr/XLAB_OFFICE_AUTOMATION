# XLAB PPTX Layout Catalog

Complete placeholder reference for all layouts in XLAB_PPT_ALL.pptx.

WHITE layout number = BLACK layout number + 15.
Placeholder indices are identical between BLACK and WHITE variants.

## Placeholder Editing Patterns

### Populating a title placeholder

Find the shape with `type="title"` or `type="ctrTitle"` in the slide XML.
Replace the empty paragraph:

```xml
<!-- Before -->
<p:txBody>
  <a:bodyPr/>
  <a:lstStyle/>
  <a:p>
    <a:endParaRPr lang="en-US"/>
  </a:p>
</p:txBody>

<!-- After -->
<p:txBody>
  <a:bodyPr/>
  <a:lstStyle/>
  <a:p>
    <a:r>
      <a:rPr lang="en-US" dirty="0"/>
      <a:t>YOUR TITLE HERE</a:t>
    </a:r>
  </a:p>
</p:txBody>
```

Titles are UPPERCASE per XLAB brand rules.

### Populating a content placeholder

Find the shape with the matching `idx` value. For multi-paragraph content:

```xml
<p:txBody>
  <a:bodyPr/>
  <a:lstStyle/>
  <a:p>
    <a:r>
      <a:rPr lang="en-US" b="1" dirty="0"/>
      <a:t>First heading</a:t>
    </a:r>
  </a:p>
  <a:p>
    <a:r>
      <a:rPr lang="en-US" dirty="0"/>
      <a:t>Body text for first section.</a:t>
    </a:r>
  </a:p>
  <a:p>
    <a:r>
      <a:rPr lang="en-US" b="1" dirty="0"/>
      <a:t>Second heading</a:t>
    </a:r>
  </a:p>
  <a:p>
    <a:r>
      <a:rPr lang="en-US" dirty="0"/>
      <a:t>Body text for second section.</a:t>
    </a:r>
  </a:p>
</p:txBody>
```

### Adding an image to a picture placeholder

1. Copy the image to `ppt/media/` in the unpacked directory
2. Add a relationship in `ppt/slides/_rels/slideN.xml.rels`:
   ```xml
   <Relationship Id="rId2" Type="http://schemas.openxmlformats.org/officeDocument/2006/relationships/image" Target="../media/your_image.png"/>
   ```
3. Transform the placeholder shape in slideN.xml from `<p:sp>` to `<p:pic>`:
   ```xml
   <p:pic>
     <p:nvPicPr>
       <p:cNvPr id="5" name="Picture Placeholder 4"/>
       <p:cNvPicPr>
         <a:picLocks noGrp="1"/>
       </p:cNvPicPr>
       <p:nvPr>
         <p:ph type="pic" sz="quarter" idx="21"/>
       </p:nvPr>
     </p:nvPicPr>
     <p:blipFill>
       <a:blip r:embed="rId2"/>
       <a:stretch><a:fillRect/></a:stretch>
     </p:blipFill>
     <p:spPr/>
   </p:pic>
   ```

**Important**: Keep the same `<p:nvPr>` with `<p:ph>` attributes from the
original placeholder so the image inherits the correct position and size
from the layout.
